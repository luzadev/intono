InTono — Privacy Policy

Ultimo aggiornamento: 27 febbraio 2026

1. Introduzione

InTono ("l'app") e' un'applicazione per l'accordatura, la pratica e l'allenamento musicale. Questa privacy policy descrive come l'app raccoglie, utilizza e protegge i tuoi dati.

2. Dati raccolti

2.1 Audio dal microfono

L'app accede al microfono del dispositivo per rilevare le note musicali suonate. L'audio viene elaborato localmente sul dispositivo in tempo reale e non viene registrato, salvato o trasmesso a server esterni.

2.2 Fotocamera e libreria foto

L'app puo' utilizzare la fotocamera o la libreria foto per scansionare immagini di spartiti musicali. Le immagini acquisite possono essere inviate a servizi di intelligenza artificiale (OpenAI o Anthropic) per il riconoscimento ottico della musica (OMR). Le immagini vengono trasmesse solo al momento della scansione e non vengono conservate sui server di terze parti oltre il tempo necessario all'elaborazione.

2.3 Dati di pratica

L'app memorizza localmente sul dispositivo le statistiche delle sessioni di pratica, i punteggi delle sfide e gli obiettivi giornalieri. Questi dati non vengono trasmessi a server esterni.

2.4 Chiavi API

Se utilizzi la funzione di scansione spartiti, l'app ti chiede di inserire una chiave API per i servizi di riconoscimento (OpenAI o Anthropic). La chiave viene memorizzata localmente sul dispositivo e trasmessa direttamente ai server del fornitore scelto solo durante le richieste di scansione.

3. Servizi di terze parti

L'app puo' comunicare con i seguenti servizi esterni esclusivamente per la funzione di scansione spartiti:

  • OpenAI API (api.openai.com) — per il riconoscimento ottico degli spartiti
  • Anthropic API (api.anthropic.com) — per il riconoscimento ottico degli spartiti

Nessun altro dato personale viene condiviso con questi servizi. Per le loro privacy policy, visita i rispettivi siti web.

4. Dati NON raccolti

L'app non raccoglie:

  • Dati di identificazione personale (nome, email, telefono)
  • Dati di localizzazione
  • Identificatori del dispositivo per scopi di tracciamento
  • Dati di utilizzo per analytics o pubblicita

L'app non contiene pubblicita e non utilizza SDK di tracciamento o analytics.

5. Conservazione dei dati

Tutti i dati dell'app (statistiche, punteggi, impostazioni) sono memorizzati esclusivamente sul tuo dispositivo. Disinstallando l'app, tutti i dati vengono eliminati.

6. Sicurezza

Le comunicazioni con i servizi di terze parti avvengono esclusivamente tramite connessioni HTTPS crittografate. I dati locali sono protetti dai meccanismi di sicurezza del sistema operativo.

7. Diritti dell'utente

Puoi eliminare tutti i dati dell'app in qualsiasi momento disinstallandola dal dispositivo. Non essendoci dati conservati su server esterni (ad eccezione delle elaborazioni temporanee per la scansione spartiti), non e' necessaria alcuna richiesta di cancellazione.

8. Modifiche alla privacy policy

Eventuali modifiche a questa privacy policy saranno pubblicate in questa pagina con la data di aggiornamento. Ti consigliamo di consultarla periodicamente.
