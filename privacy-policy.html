Privacy Policy

InTono - Accordatore e pratica per musicisti

Introduzione

La presente informativa descrive come InTono ("l'App"), sviluppata da LuZa, gestisce i dati dell'utente. La tua privacy è importante per noi e ci impegniamo a proteggerla.

Dati raccolti

InTono raccoglie e utilizza i seguenti dati esclusivamente sul tuo dispositivo:

  • Audio dal microfono: l'App accede al microfono del dispositivo per rilevare le note musicali in tempo reale (accordatore, pratica, sfida). L'audio viene elaborato localmente e non viene mai registrato, salvato o trasmesso a server esterni.
  • Fotocamera: l'App può accedere alla fotocamera per scansionare spartiti musicali tramite riconoscimento AI. Le immagini vengono inviate al servizio API selezionato (Anthropic o OpenAI) esclusivamente per il riconoscimento delle note e non vengono conservate.
  • Dati di pratica: statistiche delle sessioni (precisione, tempo, punteggi) vengono salvate localmente sul dispositivo per mostrare progressi e cronologia.
  • Preferenze: impostazioni dell'app (obiettivi giornalieri, cartella spartiti, chiave API) vengono salvate localmente sul dispositivo.

Chiavi API

Per la funzione di scansione spartiti, l'utente può inserire una propria chiave API (Anthropic o OpenAI). Questa chiave viene salvata localmente sul dispositivo in forma offuscata e viene utilizzata esclusivamente per le richieste di riconoscimento delle note. Non viene mai condivisa con terze parti.

Servizi di terze parti

L'App utilizza servizi di terze parti solo quando l'utente attiva la funzione di scansione spartiti:

  • Anthropic API o OpenAI API: per il riconoscimento delle note da immagini di spartiti. Le immagini vengono inviate tramite connessione crittografata (HTTPS). Consulta le rispettive privacy policy: Anthropic, OpenAI.

L'App non utilizza servizi di analytics, tracciamento, pubblicità o altri servizi di terze parti.

Archiviazione dei dati

Tutti i dati dell'App (statistiche, preferenze, punteggi) sono salvati esclusivamente sul tuo dispositivo. Non vengono trasmessi a server esterni. La disinstallazione dell'App rimuove tutti i dati.

Permessi richiesti

  • Microfono: necessario per il rilevamento delle note (accordatore, pratica, sfida, ear training).
  • Fotocamera: necessaria per la scansione degli spartiti.
  • Accesso ai file: necessario per importare spartiti MusicXML dal dispositivo.

Sicurezza

L'elaborazione audio avviene interamente sul dispositivo. Le comunicazioni con servizi esterni (solo per la scansione spartiti) avvengono tramite connessione crittografata HTTPS.

Minori

L'App non raccoglie consapevolmente dati personali da minori di 13 anni. L'App non richiede la creazione di account e non raccoglie dati identificativi.
